# Инструкции по тестированию

## Запуск приложения

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm start
```

3. Откройте браузер:
```
http://localhost:3000
```

## Проверка базы данных

База данных SQLite создается автоматически при первом запуске в:
```
backend/database/cybersecurity.db
```

Проверить содержимое можно с помощью SQLite Browser или командной строки:
```bash
sqlite3 backend/database/cybersecurity.db
.tables
SELECT * FROM blog_posts;
SELECT * FROM portfolio_items;
```

## Известные ограничения

1. **YouTube видео**: Для встраивания нужно вручную вводить ID видео
2. **Прогресс**: Хранится только в localStorage браузера
3. **Без авторизации**: Все данные блога и портфолио доступны всем

## Отчет о багах

Если обнаружены ошибки, проверьте:
1. Консоль браузера (F12)
2. Логи сервера в терминале
3. Сеть (Network tab) для API запросов
